# MiniReader

一个给 TrollStore/自签用的最小 iOS 阅读壳：SwiftUI + WKWebView。

- 默认打开：`https://arekert.github.io/read/`
- App 内右上角「地址」可改成你的 VPS/PWA 地址
- iOS Deployment Target：16.0
- Bundle ID：`com.yunhugy.minireader`

## GitHub Actions

推送到 `main` 后，Actions 会生成：

`MiniReader-trollstore.ipa`

下载后用 TrollStore 安装。
